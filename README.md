# 📊 Анализ IT-вакансий России (hh.ru)
### Период: май 2025 — июнь 2026

## 🎯 Цель
Анализ рынка IT-вакансий: тренды по специальностям, навыкам,
уровням зарплат и сезонной динамике. Построение регрессионной
модели прогнозирования зарплаты.

## 🔬 Гипотезы
- **Н1.** Опыт работы — ключевой фактор влияния на зарплату
- **Н2.** SQL — самый универсальный и востребованный навык
- **Н3.** Количество вакансий имеет сезонную составляющую
- **Н4.** Data Scientist показывает наибольший рост спроса

## 📈 Ключевые результаты
- Data Scientist и ML-инженер лидируют по зарплате (247К и 239К ₽)
- Факт публикации зарплаты — сильнейший предиктор (коэф. +87 380 ₽)
- Опыт работы даёт +7 464 ₽ за каждый год стажа
- R² модели = 0,86, MAE = 21 357 ₽

## 🛠️ Стек
`Python` · `BeautifulSoup` · `curl_cffi` · `pandas` · `scikit-learn`
· `matplotlib` · `seaborn` · `statsmodels`

## 🚀 Как запустить
1. Клонировать репозиторий
2. Установить зависимости:
pip install curl_cffi beautifulsoup4 pandas scikit-learn matplotlib seaborn statsmodels
3. Открыть `hh_analysis_fixed.ipynb` в Jupyter или Google Colab
